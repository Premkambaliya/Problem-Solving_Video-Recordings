# 3379. Transformed Array

**Date:** February 5, 2026
**Difficulty:** Easy

## Problem Description
You are given an integer array `nums` of length `n`.
Create an array `result` of length `n` where `result[i]` is the element `nums[j]` where `j` is the index obtained by moving `nums[i]` steps from index `i` in the array (handling circular wrap-around).

## Solution

```cpp
class Solution {
public:
    vector<int> constructTransformedArray(vector<int>& nums) {
        int n = nums.size();
        vector<int> newnums(n);
        if(n < 2){
            return nums;
        }
        for(int i=0;i<n;i++){
            if(nums[i] == 0){
                newnums[i] = 0;
            }else{
                int newindex = ((i + nums[i]) % n + n) %n;
                newnums[i] = nums[newindex];
            }
        }
        return newnums;
    }
};
```

## Complexity Analysis
- **Time Complexity:** O(N)
- **Space Complexity:** O(N)
