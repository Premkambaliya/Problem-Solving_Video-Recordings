# 3637. Trionic Array I

**Date:** February 3, 2026
**Difficulty:** Easy

## Problem Description
You are given an integer array `nums`. Return `true` if the given array is a **Trionic Array**, otherwise return `false`.
An array is **Trionic** if:
- It strictly increases, then strictly decreases, then strictly increases again.

## Solution

```cpp
class Solution {
public:
    bool isTrionic(vector<int>& nums) {
        int n = nums.size();
        int i = 0;
        while(i + 1 < n && nums[i] < nums[i+1]) i++;
        if(i == 0 || i == n-1) return false;

        while(i + 1 < n && nums[i] > nums[i+1]) i++; 
        if(i == n-1) return false;

        while(i + 1 < n && nums[i] < nums[i+1]) i++;
        if(i == n-1) return true;
        return false;
    }
};
```

## Complexity Analysis
- **Time Complexity:** O(N)
- **Space Complexity:** O(1)
