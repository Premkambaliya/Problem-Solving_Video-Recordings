# 1390. Four Divisors

**Date:** January 4, 2026  
**Difficulty:** Medium

## Problem Description
Given an integer array nums, return the sum of divisors of the integers in that array that have exactly four divisors.

## Solution

```cpp
class Solution {
public:
    int sumFourDivisors(vector<int>& nums) {
        int ans = 0;

        for (int i = 0; i < nums.size(); i++) {
            int num = nums[i];
            int count = 0;
            int sum = 0;

            for (int j = 1; j * j <= num; j++) {
                if (num % j == 0) {
                    int d1 = j;
                    int d2 = num / j;

                    if (d1 == d2) {
                        count += 1;
                        sum += d1;
                    } else {
                        count += 2;
                        sum += d1 + d2;
                    }
                }
            }
            if (count == 4) {
                ans += sum;
            }
        }

        return ans;
    }
};
```
