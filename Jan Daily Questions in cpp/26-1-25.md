# 1200. Minimum Absolute Difference

**Date:** January 26, 2025  
**Difficulty:** Easy

## Problem Description
Given an integer array `arr`, find all pairs of elements with the minimum absolute difference after sorting `arr` in non-decreasing order. Return each qualifying pair in increasing order.

## Solution

```cpp
class Solution {
public:
    vector<vector<int>> minimumAbsDifference(vector<int>& arr) {
        sort(arr.begin(), arr.end());

        int minDiff = INT_MAX;
        vector<vector<int>> result;

        for (int i = 0; i + 1 < arr.size(); i++) {
            minDiff = min(minDiff, arr[i + 1] - arr[i]);
        }

        for (int i = 0; i + 1 < arr.size(); i++) {
            if (arr[i + 1] - arr[i] == minDiff) {
                result.push_back({arr[i], arr[i + 1]});
            }
        }

        return result;
    }
};
```

## Complexity Analysis
- **Time Complexity:** O(n log n)
- **Space Complexity:** O(1) additional (excluding output storage)
