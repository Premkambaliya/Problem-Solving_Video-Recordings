# 1266. Minimum Time Visiting All Points

**Date:** January 12, 2026  
**Difficulty:** Easy

## Problem Description
On a 2D plane, there are n points with integer coordinates. Return the minimum time in seconds to visit all the points in the order given.

## Solution

```cpp
class Solution {
public:
    int minTimeToVisitAllPoints(vector<vector<int>>& points) {
        int count = 0;
        auto p1 = points[0];
        
        for(int i = 0; i < points.size(); i++) {
            auto p2 = points[i];
            int dx = p2[0] - p1[0];
            int dy = p2[1] - p1[1];
            count = count + max(abs(dx), abs(dy));
            p1 = p2;
        }
        return count;
    }
};
```

## Complexity Analysis
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
