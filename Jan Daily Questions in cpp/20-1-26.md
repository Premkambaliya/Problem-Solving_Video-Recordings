# 3314. Construct the Minimum Bitwise Array I

**Date:** January 20, 2026  
**Difficulty:** Easy

## Problem Description
You are given an array nums consisting of n prime integers. For each nums[i], find an integer ans[i] such that ans[i] | (ans[i] + 1) == nums[i].

## Solution

```cpp
class Solution {
public:
    vector<int> minBitwiseArray(vector<int>& nums) {
        vector<int> result;

        for(int &num : nums) {
            bool found = false;

            for(int x = 0; x < num; x++) {
                if((x | (x + 1)) == num) {
                    result.push_back(x);
                    found = true;
                    break;
                }    
            }
        
            if(found == false) {
                result.push_back(-1);
            }
        }
        return result;
    }
};
```
