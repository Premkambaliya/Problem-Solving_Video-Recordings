# 1458. Max Dot Product of Two Subsequences

**Date:** January 8, 2026  
**Difficulty:** Hard

## Problem Description
Given two arrays nums1 and nums2, return the maximum dot product between non-empty subsequences of nums1 and nums2.

## Solution

```cpp
class Solution {
public:
    int maxDotProduct(vector<int>& nums1, vector<int>& nums2) {
        int n = nums1.size();
        int m = nums2.size();

        vector<vector<int>> dp(n, vector<int>(m, -1e9));

        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                int take = nums1[i] * nums2[j];

                // Extend previous subsequence
                if (i > 0 && j > 0)
                    take += max(0, dp[i-1][j-1]);

                int skip1 = (i > 0) ? dp[i-1][j] : -1e9;
                int skip2 = (j > 0) ? dp[i][j-1] : -1e9;

                dp[i][j] = max({take, skip1, skip2});
            }
        }

        return dp[n-1][m-1];   
    }
};
```

## Complexity Analysis
- **Time Complexity:** O(n × m)
- **Space Complexity:** O(n × m)
