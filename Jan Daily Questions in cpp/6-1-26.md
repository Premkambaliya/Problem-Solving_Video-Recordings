# 1161. Maximum Level Sum of a Binary Tree

**Date:** January 6, 2026  
**Difficulty:** Medium

## Problem Description
Given the root of a binary tree, return the level that has the maximum sum.

## Solution

```cpp
class Solution {
public:
    int maxLevelSum(TreeNode* root) {
        int msum = INT_MIN;
        int ans = 0;

        queue<TreeNode*> que;
        que.push(root);

        int current = 1;

        while(!que.empty()) {
            int n = que.size();
            int sum = 0;
            while(n--) {
                TreeNode* temp = que.front();
                que.pop(); 
                sum += temp->val;
                if(temp->left) que.push(temp->left);
                if(temp->right) que.push(temp->right);
            }
            if(sum > msum) {
                msum = sum;
                ans = current;
            }
            current++;
        }
        return ans;
    }
};
```

## Complexity Analysis
- **Time Complexity:** O(n)
- **Space Complexity:** O(w) where w is max width
