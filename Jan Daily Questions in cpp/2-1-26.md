# 961. N-Repeated Element in Size 2N Array

**Date:** January 2, 2026  
**Difficulty:** Easy

## Problem Description
You are given an integer array nums with length 2n where there is exactly one element that is repeated n times.

## Solution

```cpp
class Solution {
public:
    int repeatedNTimes(vector<int>& nums) {
        sort(nums.begin(), nums.end());

        for(int i = 0; i < nums.size(); i++) {
            if(nums[i] == nums[i + 1]) {
                return nums[i];
            }
        }
        return -1;
    }
};
```

## Complexity Analysis
- **Time Complexity:** O(n log n)
- **Space Complexity:** O(1)
