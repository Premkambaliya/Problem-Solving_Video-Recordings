# 2943. Maximize Area of Square Hole in Grid

**Date:** January 15, 2026  
**Difficulty:** Medium

## Problem Description
You are given a grid with horizontal and vertical bars. Find the maximum square area that can be created by removing consecutive bars.

## Solution

```cpp
class Solution {
public:
    int maximizeSquareHoleArea(int n, int m, vector<int>& hBars, vector<int>& vBars) {
        sort(hBars.begin(), hBars.end());
        sort(vBars.begin(), vBars.end());

        int maxiH = 1, maxiV = 1, maxi = 1;

        for (size_t i = 1; i < hBars.size(); ++i) {
            if (hBars[i] == hBars[i - 1] + 1) {
                maxi++;
            } else {
                maxiH = max(maxiH, maxi);
                maxi = 1;
            }
        }
        maxiH = max(maxiH, maxi);

        maxi = 1;

        for (size_t i = 1; i < vBars.size(); ++i) {
            if (vBars[i] == vBars[i - 1] + 1) {
                maxi++;
            } else {
                maxiV = max(maxiV, maxi);
                maxi = 1;
            }
        }
        maxiV = max(maxiV, maxi);

        int side = min(maxiH + 1, maxiV + 1);
        return side * side;
    }
};
```

## Complexity Analysis
- **Time Complexity:** O(n log n + m log m)
- **Space Complexity:** O(1)
